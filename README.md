# lfp-jec
Jec是一个基于微服务架构的，分布式Java EE 企业级快速开发平台，致力于打造大型企业分布式、互联网、云开发平台，基于代码重用、组件重用、业务逻辑重用、组装重用，结合在线流程设计器，将开发人员从传统的流程管理业务开发中解放出来，把更多的精力集中解决客户的业务数据处理，为企业提供安全、稳定、统一、服务化的框架。
